## ⚙️ 개발 환경
- **프로그래밍 언어**: Java 17
- **IDE**: IntelliJ IDEA
- **운영 체제**: Windows

## 📌 주요 기능
- **기본 사칙 연산**: 덧셈, 뺄셈, 곱셈, 나눗셈
- **입력 검증**: 잘못된 입력에 대한 예외 처리
- **캡슐화**: 데이터 접근 제한을 위한 Getter와 Setter 메서드 사용
- **사용자 친화적인 메시지 출력**: 에러 메시지와 결과 출력
